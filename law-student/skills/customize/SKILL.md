# /配置定制

## 触发场景

用户输入 `/law-student:配置定制`。他们想修改学习配置中的某项内容——
课程、学习风格偏好、法考备考科目、材料路径——无需重新运行完整冷启动访谈。

## 执行步骤

1. **读取配置。** 读取 `~/.claude/plugins/config/claude-for-legal/law-student/CLAUDE.md`。
   如不存在或含 `[PLACEHOLDER]` 值，提示先运行 `/law-student:冷启动访谈`。

2. **展示可定制项目总览：**

   - **个人信息** — 姓名、就读院校、年级（大一至研三）、法学院
   - **当前学期** — 正在修的课程、学分、教学大纲路径
   - **法考备考** — 考试年份、目标科目（客观题/主观题）、备考阶段、使用的培训机构/资料（瑞达/厚大/众合/柏杜/自学）
   - **弱势科目** — 各科目当前掌握程度（很弱/较弱/一般/较好/很好）
   - **学习风格** — 研习型/讲解型、视觉/听觉/阅读/动手偏好
   - **知识体系** — 笔记偏好（详细/思维导图/大纲）、知识体系文件路径
   - **训练频率** — 训练节奏、闪卡复习频率（艾宾浩斯桶间隔）
   - **种子材料** — 已上传的笔记、大纲、历年试卷、教材清单

3. **询问修改内容。**

4. **执行修改。** 展示当前值、询问新值、说明影响（如"将民法掌握度从'一般'改为'较弱'→学习计划和训练将增加民法题量"）、确认、写入配置。

5. **收尾。** "完成。您的下一次输出将反映此变更。还有其他需要调整的吗？"

## 常见定制场景

| 场景 | 用户可能说 | 需要调整的配置项 |
|---|---|---|
| 新学期开学 | "新学期了"、"课程变了" | 当前课程列表、教学大纲、教材 |
| 法考备考启动 | "开始准备法考了" | 考试年份、备考阶段、培训机构 |
| 某科目进步了 | "民法现在掌握得不错了" | 科目掌握度 → 学习计划自动调权 |
| 换备考机构 | "换厚大的课了" | 备考机构、教材、题目来源 |
| 学习风格调整 | "还是先讲再练吧" | 研习型 → 讲解型 |

## 护栏规则

- **绝不删除科目。** 如要"不考某科目"，标记为 `[当前不在备考范围]` 而非删除。法考8大科目均为必考——如果学生说"不考三国法"，提醒：法考所有科目均为必考，不能跳过。
- **标记内部不一致。** 如配置显示"法考备考"但只填写了3个科目的掌握度——标记遗漏。
- **标记不合规行为。** 如配置显示学生在处理真实法律事务——提醒法学生不能提供法律意见。
- **每次只改一项。** 不要变成完整的冷启动访谈。
