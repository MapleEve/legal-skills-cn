# /cold-start-interview

1. 检查 `~/.claude/plugins/config/claude-for-legal-cn/law-student/CLAUDE.md`。如已填充且无 `--redo`，覆盖前确认。如有缓存配置则迁移。
2. 使用以下访谈工作流。
3. 遍历：第0部分（使用者身份/已连接内容——学生vs毕业生vs其他；文档存储可用性）、第1部分（现在在哪——大一至研三、法学院、当前课程）、第2部分（学习方式——研习型vs讲解型、视觉/听觉/阅读/动手）、第3部分（强项/薄弱/回避的科目——8大法考科目+法学院核心课程）、第4部分（材料采集——目标至少10项：教学大纲、课堂笔记、知识体系、案例摘要、闪卡、历年试卷、评分文章、法考练习题集、教材）。
4. 重读捕获的答案。捕捉矛盾、偏差、值得命名的缺口。
5. 写入配置文件，包含使用者身份和可用集成。如分享材料少于10项则添加 `数据有限` 标记。
6. 与用户确认。

## 目的

了解中国法学生是谁、在哪、学什么、怎么学、有什么材料、准备什么考试。写入配置，使其他技能按学生实际校准，而不是按通用模板运行。

## 冷启动检查

按文件存在/占位符/暂停/已填充四种状态分派。

## 检查共享配置

查找共享公司配置（如存在）。对于法学生，通常没有公司配置——直接进入学生特定问题。

## 访谈节奏

- 假设答案存在于某处——提示粘贴链接或文件
- 每条消息2-3个可回答的提示
- 支持暂停和恢复（说"暂停"可中断，再次运行从断点恢复）

## 访谈

### 开篇

> 我准备好了——先了解你的情况：在哪读法学院、什么年级、当前学什么、准备什么考试、有什么学习材料。回答越具体，每个技能就越能按你的实际情况工作。准备好了吗？

### 第0部分：使用者身份

> 首先，你现在是哪个阶段的法学学习者？
>
> 1. **法学生（本科）**——大一至大四
> 2. **法学生（研究生）**——法学硕士/法律硕士（法学）/法律硕士（非法学）/法学博士
> 3. **法考备考者**——已毕业，正在准备法考
> 4. **其他**——请说明

> 你偏好哪种学习交互方式？
>
> 1. **研习型**——喜欢被提问、被追问，先自己思考再看答案。我问你答，答错了再解释。
> 2. **讲解型**——喜欢先被解释概念/法条，然后再做题测试。我先讲你再练。

> 你有可用的文件存储吗？（钉钉文档/飞书文档/企业微信/本地文件夹/Notion/其他）学习材料从哪里读取最方便？

### 第1部分：现在在哪

> 告诉我你现在的位置：
>
> - **就读院校：** 哪所法学院？
> - **年级：** 大一/大二/大三/大四/研一/研二/研三
> - **当前学期课程：** 这学期在修哪些课？
> - **是否有考试在即：** 法考（哪年？客观题/主观题？）/ 法学院期末考试（哪些科目？）

> 如果正在准备法考：
> - 目标考试年份？
> - 目前备考阶段（零基础起步/有基础系统复习/冲刺）？
> - 正在使用的备考机构/资料（瑞达/厚大/众合/柏杜/自学）？

### 第2部分：学习方式

> 除了研习型vs讲解型，你还有什么学习偏好？
>
> - 学习新知识的偏好：先看视频课程 / 先读教材 / 先做题倒逼学习
> - 笔记风格：详细记录 / 思维导图 / 知识体系大纲 / 基本不记
> - 学习环境：独自学习 / 小组讨论 / 混合

### 第3部分：强项/薄弱/回避

> 对中国法8大法考科目（或你当前学的科目），你的自我评估如何？
>
> 逐科目告诉我：很弱 / 较弱 / 一般 / 较好 / 很好

| 科目 | 自评 | 备注（如有） |
|---|---|---|
| 民法 | ? | |
| 刑法 | ? | |
| 行政法与行政诉讼法 | ? | |
| 民事诉讼法 | ? | |
| 刑事诉讼法 | ? | |
| 商经知 | ? | |
| 理论法（法理/宪法/法制史/职业道德） | ? | |
| 三国法（国际法/国际私法/国际经济法） | ? | |

> 有没有你一直在回避的科目？那些知道该学但就是不想碰的——告诉我。

帮助学生正视回避倾向——回避的科目通常是最需要帮助的。

### 第4部分：材料采集

> 这是我需要的最重要的东西——你已有的学习材料。越多越好，目标至少10项。

请求上传或分享：
- 教学大纲（当前学期/备考科目）
- 课堂笔记
- 已有知识体系/复习大纲
- 案例摘要（已做过的case brief）
- 闪卡牌组（如有）
- 历年试卷（上学期/往年）
- 评分文章/被批改过的作业
- 法考练习题集/真题集
- 教材信息（书名/作者/出版社/版本）
- 老师的课件/讲义（如有权限获取）

> 提醒：材料越多，生成的题目和反馈越精准。比如有真实的历年试卷，预测考试就会更准确。

## 法学领域真实事项检查

如果用户在访谈中提到正在处理真实案件、提供真实法律咨询或以律师身份活动——提醒：
- 法学生不得提供法律意见（《律师法》禁止非律师从事法律服务）
- 本插件仅用于学习训练，不能替代执业律师

## 撰写配置后

展示插件能力：
- `考试预测` — 基于历年试卷预测考点
- `课堂提问准备` — 预测老师的课堂提问并训练
- `闪卡` — 艾宾浩斯记忆法管理法条记忆
- `案例分析写作训练` — 给案例分析答案评分
- `法学写作` — 给法学写作草稿反馈
- `学习训练` — N道题集中训练
- `学习计划` — 法考/期末备考计划

提出首次任务建议。标记缺口（特别是材料不足的科目）。以可修改性说明收尾。

## 本访谈不做的事

- 不读取学生个人对话历史
- 不假设学生的学习风格或能力水平
- 不跳过材料采集——材料是精准度的基础
- 不推广特定法考培训机构
