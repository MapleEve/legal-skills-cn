# /employment-legal:leave-tracker

检查所有具有硬性法定截止日期的开放假期，仅揭示需要决定或行动的事项。不是状态看板——告诉您需要做什么以及为什么。

## 指示

1. 加载`假期追踪`代理并运行完整工作流。

2. 如果没有HR系统连接且不存在`~/.claude/plugins/config/claude-for-legal-cn/employment-legal/leave-register.yaml`，提示律师上传假期表格或使用`/employment-legal:log-leave`添加条目。

3. 仅针对需要行动的假期发出提醒。正常进行中的假期每行一句话概括。

4. 追踪结束时以决策树收尾——告知下一步行动选项。

## 示例

```
/employment-legal:leave-tracker
```

每周运行一次——设置周一早上提醒来调用`/employment-legal:leave-tracker`。

## 中国法下假期制度详解

### 一、法定年假

**法律依据**：《职工带薪年休假条例》（国务院令第514号）、《企业职工带薪年休假实施办法》（人社部令第1号）

| 累计工作年限 | 年假天数 | 法律依据 |
|---|---|---|
| 1年 ≤ 工龄 < 10年 | 5天 | 《职工带薪年休假条例》第3条 |
| 10年 ≤ 工龄 < 20年 | 10天 | 同上 |
| 工龄 ≥ 20年 | 15天 | 同上 |

**关键规则：**
- 国家法定休假日、休息日不计入年假假期（《条例》第3条）
- 单位因工作需要不能安排职工休年假的，经职工本人同意，可以不安排。对应休未休的年假天数，按日工资收入的300%支付报酬（《条例》第5条）
- 用人单位安排职工休年假，但职工因本人原因且书面提出不休的，用人单位只支付正常工作期间的工资收入（《实施办法》第10条）
- **年假一般不跨年度安排**。单位因生产、工作特点确有必要跨年度安排的，可以跨1个年度安排（《条例》第5条）——即年假跨年结转最多延至次年年底
- 不享受当年年假的情形（《条例》第4条）：享受寒暑假且天数多于年假、请事假累计20天以上且不扣工资、病假累计满相应月数

**追踪要点：**
- 年初总天数
- 已休天数（含已审批未休）
- 剩余天数
- 跨年结转截止日（次年12月31日前必须安排完毕）
- 离职时的年假结算（应休未休折算天数 = 当年度在单位已过日历天数÷365×全年应享受天数 - 当年度已休天数）

### 二、医疗期（病假）

**法律依据**：《企业职工患病或非因工负伤医疗期规定》（劳部发[1994]479号）

| 实际工作年限 | 本单位工作年限 | 医疗期 | 累计病休时间计算周期 |
|---|---|---|---|
| < 10年 | < 5年 | 3个月 | 6个月内 |
| < 10年 | ≥ 5年 | 6个月 | 12个月内 |
| ≥ 10年 | < 5年 | 6个月 | 12个月内 |
| ≥ 10年 | 5-10年 | 9个月 | 15个月内 |
| ≥ 10年 | 10-15年 | 12个月 | 18个月内 |
| ≥ 10年 | 15-20年 | 18个月 | 24个月内 |
| ≥ 10年 | ≥ 20年 | 24个月 | 30个月内 |

**病假工资**：不得低于当地最低工资标准的80%（劳部发[1995]309号第59条、各地可能有更高标准）。

**特殊疾病**（癌症、精神病、瘫痪等）：经企业和劳动主管部门批准，可适当延长医疗期。

**医疗期满后的处理：**
- 能从事原工作→恢复原岗位
- 不能从事原工作→调岗
- 调岗后仍不能从事→用人单位可解除合同（须支付N+1补偿金，《劳动合同法》第40条第一项）

**追踪要点：**
- 医疗期起算日期
- 累计病休天数（在计算周期内的）
- 医疗期届满日期
- 病假工资发放标准
- 医疗期满前的返岗评估

### 三、产假

**法律依据**：《女职工劳动保护特别规定》（国务院令第619号）+ 各省人口与计划生育条例

| 项目 | 天数 | 法律依据 |
|---|---|---|
| 基本产假 | 98天（产前可休15天） | 《特别规定》第7条 |
| 难产 | +15天 | 《特别规定》第7条 |
| 多胞胎（每多1个） | +15天 | 《特别规定》第7条 |
| 流产（< 4个月） | 15天 | 《特别规定》第7条 |
| 流产（≥ 4个月） | 42天 | 《特别规定》第7条 |
| 地方奖励假 | 各地不同（30-80天） | 各省人口与计划生育条例 |

**地方奖励假举例：**
- 北京：+60天（《北京市人口与计划生育条例》2021修正）
- 上海：+30天（《上海市人口与计划生育条例》）
- 广东：+80天（《广东省人口与计划生育条例》）
- 浙江：+60天
- 江苏：+60天

**产假期间待遇：**
- 已参加生育保险：生育津贴由生育保险基金支付（按单位上年度职工月平均工资）
- 未参加：由用人单位按产假前工资标准支付

**追踪要点：**
- 预产期/产假起算日期
- 产假届满日期（98天+地方奖励+难产/多胞胎附加）
- 生育津贴申请状态
- 返岗日期
- 哺乳期权益（每天1小时哺乳时间，至婴儿满1周岁）

### 四、哺乳期/哺乳假

- 每天1小时哺乳时间，至婴儿满1周岁（《特别规定》第9条）
- 多胞胎的，每多哺乳1个每天增加1小时
- 女职员工劳动时间内哺乳往返途中的时间计入劳动时间

### 五、婚假

各地不同，按各省人口与计划生育条例：
- 国家法定：3天（原《关于国营企业职工请婚丧假和路程假问题的规定》）
- 多数省份在法定3天基础上增加奖励假：北京10天、上海10天、广东3天等

**追踪要点：**
- 婚假起止日期
- 是否在领取结婚证后规定期限内申请（部分企业有内部规定）

### 六、丧假

- 直系亲属（父母、配偶、子女）：1-3天
- 岳父母/公婆：部分企业参照执行
- 外地奔丧可另给路程假

### 七、陪产假

各地不同（各省人口与计划生育条例）：
- 北京：15天、上海：10天、广东：15天、浙江：15天

### 八、探亲假

《国务院关于职工探亲待遇的规定》（1981年，适用面窄——国家机关、人民团体、全民所有制企业事业单位）：
- 探望配偶（两地分居）：每年1次，30天
- 未婚探望父母：每年1次，20天
- 已婚探望父母：每4年1次，20天

## 工作流

### 步骤1：加载假期登记册

从`~/.claude/plugins/config/claude-for-legal-cn/employment-legal/leave-register.yaml`（或HR系统接口）加载当前所有活跃假期。

### 步骤2：按假期类型分类

逐条检查每条活跃假期，按以下维度分类：

| 状态 | 含义 |
|---|---|
| ✅ 正常 | 假期进行中，无即将到期事项，无需行动 |
| ⚠️ 即将到期 | 未来14天内有关键截止日（返岗日、年假跨年结转截止、医疗期满评估、产假期满） |
| 🔴 已逾期/异常 | 已过关键截止日、超期未返岗未说明、假期天数计算有误 |
| 📋 需关注 | >14天但<30天有关键截止日——记录但非紧急 |

### 步骤3：按管辖应用法定规则

根据员工劳动合同履行地（省市），应用对应的法定假期天数、医疗期时长、最低病假工资等地方性规定。

### 步骤4：产出行动清单

仅列出需要行动的条目。格式：

```
⚠️ [员工姓名] - [假期类型] - 截止日[date] - 行动：[具体事项]
🔴 [员工姓名] - [假期类型] - 问题描述 - 行动：[紧急处理事项]
📋 [员工姓名] - [假期类型] - 截止日[date] - 提示：[关注事项]
```

## 以下一步决策树结束

以CLAUDE.md`## 输出`中的下一步决策树结束。将选项定制为假期追踪结果——五个默认分支为起点，非锁定。决策树即输出；律师/HR选择。

## 此技能不做什么

- 不替代HR系统。这是基于登记册的缩影，非正式人事记录。
- 不替代企业对请假流程的内部审批。仅追踪法定权利和截止日。
- 不做出"应当批准/不批准"的决定。
- 不计算复杂的工龄合并（涉及多段工作经历时）。
- 不替代劳动法律师对医疗期满解除、孕期/产期/哺乳期"三期"女职工保护等复杂问题的法律判断。
