# 投后整合管理

## 中国投后整合法律框架

在中国法律环境下，投后整合涉及以下关键法律步骤：

### 工商变更（交割后首要任务）
- **股东变更登记** — 向市场监督管理局申请股东变更登记（《公司法》+《市场主体登记管理条例》）
- **法定代表人变更** — 如涉及，需修改章程并登记
- **董监高备案** — 新任董事、监事、高级管理人员的备案
- **章程备案** — 修订后的公司章程
- **外商投资信息报告变更** — 如涉及外资（《外商投资法》）

### 其他行政手续
- **税务登记信息变更** — 向主管税务机关办理
- **外汇登记变更** — 外资企业涉及
- **行业资质变更** — 如公司持有需登记的经营资质
- **银行账户变更** — 预留印鉴/授权签字人变更

### 合同层面
- **控制权变更条款触发** — 筛查所有重大合同中控制权变更条款
- **对方同意/通知** — 获取必要的对方同意或发送变更通知
- **合同转让** — 如交易结构涉及合同主体变更
- **知识产权备案变更** — 商标、专利、域名的权利人变更登记

### 人员层面
- **劳动合同承继** — 股权收购下，劳动合同继续有效
- **社保公积金** — 确保连续缴纳
- **竞业限制和保密** — 审查并更新关键人员的协议

---

## 目的

外部律师完成交易交割。企业法务/投后管理团队接手后续。本技能是投后整合的项目管理层 — 不是业务整合，不是IT系统迁移，不是HR组织设计。法务工作流：工商变更、合同同意、主体变更、知识产权备案、收购协议义务跟踪。

## 中国投后整合分阶段计划

### D1（交割日）
- [ ] 签署交割文件（交割备忘录、资金划转凭证）
- [ ] 确认公章、营业执照、财务章等已交接
- [ ] 确认银行账户控制权已移交
- [ ] 确认关键证照、资质原件已交接

### D1-D7（交割后第一周）
- [ ] 启动工商变更登记申请
- [ ] 向关键合同对方发送控制权变更通知
- [ ] 启动需要对方同意的合同函件
- [ ] 召开新股东会/董事会并形成纪要
- [ ] 更新银行预留印鉴和授权人

### D7-D30（交割后第一个月）
- [ ] 完成工商变更登记并取得新营业执照
- [ ] 完成税务登记信息变更
- [ ] 完成外汇登记变更（如适用）
- [ ] 启动商标/专利/域名权利人变更
- [ ] 完成员工沟通和关键人员协议更新
- [ ] 投保新的董事责任险（如适用）

### D30-D90（交割后第三个月）
- [ ] 完成所有控制权变更同意的获取或跟踪
- [ ] 完成知识产权备案变更
- [ ] 完成供应商/客户合同变更通知
- [ ] 完成资质/许可的变更或重新申请
- [ ] 目标公司合规体系与买方对接

### D90-D180（交割后半年）
- [ ] 完成遗留合同问题的解决
- [ ] 完成主体精简/合并（如计划中）
- [ ] 收购协议义务的年度审查
- [ ] 整合效果评估

## 模式

### 模式1：初始化 — 从收购协议和交割清单构建
以收购协议、交割清单和交易摘要为基础，生成分阶段工作计划和同意追踪表。

### 模式2：合同转让 — 导入合同列表
以尽调中识别的合同清单为输入，筛查需要对方同意的合同、需要通知的合同、需办理转让手续的合同。按紧迫程度分层：
- **T1 — 交割后立即处理：** 核心业务合同（主要客户/供应商/银行）
- **T2 — 一个月内处理：** 重要但非紧急
- **T3 — 三个月内处理：** 常规合同
- **T4 — 无需特殊处理：** 标准化合同/无需通知

## 此技能不做什么

- 不管理业务整合（销售团队合并、产品线整合等）。仅追踪法务在这些流程中的接触点。
- 不起草同意请求函或合同变更协议。识别需要什么；起草由律师完成。
- 不就赔偿索赔或收购协议违约提供法律建议。
- 不追踪盈利能力表现或协同效应实现。
