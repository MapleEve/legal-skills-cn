# 主体合规

## 中国主体合规法律框架

依据以下法规管理企业主体合规义务：

- **《公司法》(2024)** — 公司设立、变更、注销基本规范
- **《市场主体登记管理条例》** — 各类市场主体的登记管理
- **《企业信息公示暂行条例》** — 年度报告公示制度
- **《外商投资法》** — 外资企业额外合规要求
- **《税务登记管理办法》** — 税务登记与变更
- **《外汇管理条例》** — 外汇登记（外资企业适用）

主要监管机构：

| 监管机构 | 管辖事项 |
|---|---|
| 市场监督管理局 | 工商登记、变更、注销、年度报告公示、经营异常名录 |
| 税务局 | 税务登记、变更、注销、纳税申报 |
| 外汇管理局 | 外汇登记、变更（外资企业） |
| 商务部 | 外资企业备案/审批（特定行业） |
| 地方金融监管局 | 地方金融类企业额外要求 |

---

## 目的

每个主体在每个工商登记地都有年度报告、信息变更和备案义务。错过截止日期的后果各有不同 — 从列入经营异常名录到罚款到注销限制。本技能维护一个追踪器，知道什么到期、何时到期、针对哪个主体。

## 追踪器文件

位于 `~/.claude/plugins/config/claude-for-legal/corporate-legal/entities/compliance-tracker.yaml`。

## 核心合规事项

### 年度报告公示
- **申报主体：** 所有企业（有限责任公司/股份有限公司/非公司企业法人/合伙企业/个人独资企业/分支机构）
- **申报时间：** 每年1月1日至6月30日
- **申报内容：** 企业基本信息、股东及出资信息、资产状况信息、对外担保信息等
- **申报渠道：** 国家企业信用信息公示系统 (www.gsxt.gov.cn)
- **逾期后果：** 列入经营异常名录；满3年仍未履行的，列入严重违法失信企业名单

### 工商变更登记
- **触发情形：** 名称、住所、法定代表人、注册资本、经营范围、股东/股权结构、经营期限等发生变更
- **时限：** 变更决议或决定作出之日起30日内（一般情形）
- **程序：** 内部决议 → 修改章程 → 向登记机关申请变更登记 → 换发营业执照

### 备案
- **清算组备案：** 解散事由出现之日起15日内
- **分公司备案：** 设立/变更/注销
- **外资企业：外商投资信息报告（初始报告 + 变更报告 + 年度报告）

### 注销
- **简易注销：** 未开业或无债权债务企业，公告期20日
- **一般注销：** 清算组备案 → 清算公告45日 → 清算报告 → 注销登记
- **税务注销：** 先税务后工商，《税收征收管理法》规定

## 模式

### 模式1：初始化 — 从主体表创建追踪器
以用户提供的主体清单（含主体名称、统一社会信用代码、类型、登记机关、成立日期等）为基础，构建合规追踪器，自动推算各项申报截止日期。

### 模式2：报告 — 揭示即将到期和逾期项目
展示未来30/60/90天内的申报截止日期。标注：
- 🔴 已经逾期
- 🟠 30天内到期
- 🟡 60天内到期
- 🟢 90天内到期

### 模式3：更新 — 记录完成状态
手动记录或通过代理报告文件更新。

### 模式4：审计 — 全面健康审计
从以下维度审计：
- 年度报告是否按期完成
- 变更登记是否与实际情况一致
- 备案是否完整（外资企业额外核查）
- 经营异常名录/严重违法失信名单查询
- 存续状态核查（吊销/注销/迁出）

### 模式5：导出 — CSV或表格格式

## 外资企业额外合规

依据《外商投资法》及其实施条例：
- 外商投资信息报告制度（替代原商务备案）
- 负面清单管理制度 — 特定行业需审批
- 国家安全审查 — 特定行业外商投资须通过
- 外汇登记管理

## 此技能不做什么

- 不代表企业提交任何申报。输出是追踪器和待办列表；申报由企业人员或代理完成。
- 不拉取存续证明。追踪上次确认时间；获取证明需手动或通过代理。
- 不决定是否需要商务/国安审查。该分析取决于企业业务活动事实，需律师确认。
