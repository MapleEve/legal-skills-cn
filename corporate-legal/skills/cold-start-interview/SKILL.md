# 冷启动访谈

## 中国公司法背景

本技能依据中国公司法框架。《公司法》(2024) 核心结构：

```
股东会（最高权力机构）
  ├─ 董事会（经营决策机构，三人以上）
  ├─ 监事会（监督机构，三人以上）
  └─ 经理（由董事会聘任/解聘）

公司类型：有限责任公司 / 股份有限公司
三会一层：股东会 + 董事会 + 监事会 + 高级管理层
```

关键监管机构：市场监督管理局（工商登记）、证监会（上市公司监管）、交易所（信息披露）、税务局、外汇管理局（外资）。

---

## 目的

公司法务角色差异极大。一个50人创业公司的单一法务总监同时负责投资并购、管理股权结构表和担任董事会秘书。一个上市公司的公司法务可能只负责信息披露委员会流程。本访谈找出哪些领域对用户是活跃的并仅构建相关的执业画像。

## 模块选择

> 以下哪些是你的常法工作的一部分？
>
> 1. **投资并购** — 交易：购买、出售、投资或剥离业务/股权
> 2. **董事会&秘书** — 三会会议准备、纪要、决议、工商备案
> 3. **上市公司** — 证监会/交易所信息披露、披露委员会、内幕交易防控
> 4. **主体管理** — 子公司管理、工商登记、股权结构表、年度申报

每项选中的模块触发针对性后续问题。

## 公司画像

- 公司类型：有限责任公司 / 股份有限公司
- 公司治理结构：三会一层的实际运作情况
- 所属行业：是否涉及特殊监管行业（金融、医疗、教育等）
- 发展阶段：初创/成长/Pre-IPO/上市/成熟
- 外资成分：《外商投资法》是否适用

## 各模块访谈

### 投资并购模块（如活跃）
- 尽调实质性阈值（金额、类型）
- 交易类型偏好（股权收购/资产收购/合并/少数股权投资）
- 问题备忘录格式偏好
- 数据室管理方式
- 交易团队构成和汇报线
- 交割清单模板

### 董事会&秘书模块（如活跃）
- 公司类型决定治理结构：有限公司 vs 股份公司
- 会议纪要格式（全文叙述/决议摘要/混合）— 从种子纪要学习
- 书面决议/传签决议的适用情形
- 章程中的表决机制特殊约定
- 会议召集和通知程序
- 工商登记变更的内部流程

### 上市公司模块（如活跃）
- 上市交易所（上交所主板/科创板、深交所主板/创业板、北交所、港交所）
- 财务年度
- 信息披露委员会（组成和运作机制）
- 内幕信息知情人登记制度
- 定期报告和临时公告的披露节奏
- 关联交易审议和披露流程

### 主体管理模块（如活跃）
- 主体列表和工商登记信息
- 年度报告公示管理
- 经营异常名录查询机制
- 税务登记和纳税申报
- 外资企业额外合规要求（如适用）
