# /侵权警告函

两种模式。选一个：

- `/ip-legal:侵权警告函 --send`——起草根据维权立场校准的知识产权侵权警告函/律师函。发送前运行响亮关卡。
- `/ip-legal:侵权警告函 --receive`——对收到的侵权警告函进行分诊。产生带建议的结构化选项备忘录。

## 指示

1. 加载执业画像配置。
2. 检查案件工作空间。
3. 根据选定模式调度发送或接收工作流。
4. 尊重关卡。在发送模式中，发送前运行响亮关卡。

## 中国知识产权法下的侵权警告函

侵权警告函（律师函）是权利人主张知识产权权利的常用前置手段。依据中国法律：

### 法律依据
- **商标**：《商标法》(2019)第57条（侵权行为认定）、第60条（行政查处）、第63条（损害赔偿）
- **专利**：《专利法》(2020)第11条（侵权行为）、第65条（许可使用费参照）、第71条（赔偿计算）
- **著作权**：《著作权法》(2020)第52-53条（侵权行为）、第54条（赔偿计算）
- **反不正当竞争**：《反不正当竞争法》关于混淆行为、侵犯商业秘密等的规定

### 发送警告函的特殊法律风险
**1. 确认不侵权之诉（《最高人民法院关于审理侵犯专利权纠纷案件应用法律若干问题的解释》第18条）**
专利权人发送警告函后，被警告方或利害关系人可以书面催告权利人行使诉权。权利人自收到书面催告之日起一个月内或自书面催告发出之日起二个月内，不撤回警告也不提起诉讼的，被警告方可向人民法院提起确认不侵权之诉。**商标/著作权领域亦有类似风险。**

**2. 专利稳定性评估（发送前必须）**
发送专利侵权警告函前，应评估专利权的稳定性：
- 实用新型和外观设计专利应取得专利权评价报告
- 评估是否存在现有技术/现有设计抗辩
- 发送警告函后若专利被宣告无效，可能构成错误通知，承担赔偿责任

**3. 商业诋毁风险（《反不正当竞争法》第11条）**
警告函若向社会公众（如向被警告方的客户或经销商）广泛散发，可能被认定为商业诋毁。发送范围应合理限制。

**4. 知识产权海关保护（《知识产权海关保护条例》）**
权利人可以将其知识产权向海关总署备案（有效期10年，可续展）。备案后发现侵权嫌疑货物即将进出口的，可向货物进出境地海关申请扣留侵权嫌疑货物。警告函可提及海关备案和扣货申请的可能性。

## 发送模式——起草侵权警告函

### 步骤1：识别权利
- 著作权：作品类型、创作日期、登记号（如有）
- 商标权：注册号、核定类别、商品/服务、有效期
- 专利权：专利号、类型、名称、授权日、有效性状态、专利权评价报告
- 其他：商业秘密、企业名称、包装装潢等

### 步骤2：识别侵权行为
- 具体行为：制造/销售/使用/进口/许诺销售
- 侵权产品/服务描述
- 侵权证据：截图、购买记录、公证材料
- 侵权范围：地域、规模、持续时间

### 步骤3：识别双方当事人关系
- 是否为竞争对手？
- 对方是否为平台内经营者（可同时适用《电子商务法》第42-43条平台投诉）？
- 是否存在在先许可、合作或授权关系？
- 对方是否在供应链中（制造商/经销商/终端用户）？

### 步骤4：确定诉求
- **停止侵权**：立即停止制造、销售、许诺销售、使用等侵权行为
- **消除影响**：在侵权行为影响的范围内消除影响
- **赔偿损失**：明确金额或计算方式
- **销毁侵权产品**：销毁库存侵权产品及专门用于制造侵权产品的模具
- **提供信息**：要求披露侵权产品来源、销售数量、销售金额

### 步骤5：专利稳定性前置评估（仅专利案件）
**发送前必备步骤：**
- 核查专利年费缴纳状态
- 实用新型/外观设计专利应已取得专利权评价报告
- 评估现有技术抗辩可能性
- 评估专利被宣告无效的风险

如为实用新型或外观设计专利且无专利权评价报告 → **必须标注**专利权评价报告状态，建议取得评价报告后再发送。

### 步骤6：按维权立场校准函件力度
- **取证式**（轻）：以停止侵权为主要目的，保留后续维权权利
- **谈判式**（中）：表达协商意愿，提出许可或和解方案
- **最后通牒式**（重）：设定最后期限，明确告知将要启动的法律程序

### 步骤7：起草
按标准结构撰写：
1. **函头**：律所/公司抬头、函号、日期
2. **委托人及权利证明**：权利主体、权利证书信息
3. **侵权事实**：时间、地点、行为、产品/服务、证据
4. **法律依据**：引用具体法律条文
5. **侵权认定**：对比分析
6. **具体要求**：逐项列明
7. **履行期限**：明确日期
8. **后续措施告知**：行政投诉/法院起诉/海关扣货/平台投诉
9. **保留权利声明**

### 步骤8：发送前响亮关卡
**发送警告函可能触发：**
- 确认不侵权之诉（对方可主动起诉）
- 专利权无效宣告（对方反击）
- 商业诋毁反诉（如发送范围不当）

**关卡检查清单：**
- [ ] 权利证明真实有效
- [ ] 侵权行为证据确实充分（公证固定为佳）
- [ ] 专利稳定性已评估（实用新型/外观设计已取得评价报告）
- [ ] 发送对象适当（直接影响方而非广泛散发）
- [ ] 函件内容无商业诋毁风险
- [ ] 已评估对方反诉/无效宣告风险

### 步骤9：输出

## 接收模式——对收到的侵权警告函分诊

### 步骤1：读取函件和证据
- 发件人身份和授权
- 主张的权利及其有效性
- 指控的具体行为
- 附带的证据材料
- 要求的期限和后果

### 步骤2：评估权利主张
| 维度 | 分析要点 |
|---|---|
| **权利有效性** | 商标注册状态/专利年费/专利权评价报告/可被无效的可能性 |
| **权利范围** | 商标核定类别/专利权利要求保护范围/著作权作品独创性范围 |
| **侵权比对** | 被控行为是否落入权利保护范围 |
| **证据质量** | 证据是否经公证固定/证据链是否完整 |

### 步骤3：评估己方风险和抗辩
- 是否构成侵权（全面分析）
- 可用的抗辩：现有技术抗辩/合理使用/在先使用/合法来源/权利无效
- 潜在赔偿金额范围
- 业务影响评估（是否需要停产/停售）

### 步骤4：应对选项

1. **遵守/停止侵权**：侵权成立，风险高——停止被控行为
2. **谈判/许可**：侵权可能成立——启动许可谈判或和解谈判
3. **坚定回应/说明不侵权理由**：侵权不成立——以事实和法律依据坚定回应
4. **主动提起确认不侵权之诉**：函件影响重大且己方不侵权——主动提起诉讼确认不侵权（《专利法》相关司法解释）
5. **提起专利权无效宣告**：对方专利权稳定性差——向国知局请求宣告专利无效
6. **忽略/暂不回应**：仅在函件明显无依据时——有风险，一般不建议

### 步骤5：截止日期分析和行动
- 对方设定的履行期限
- 是否已超过期限
- 距离期限还有多少时间
- 是否需要申请延期
- 对方可能启动的程序和时间线

### 步骤6：撰写分诊备忘录

## 输出格式

### 发送模式——侵权警告函草稿
```markdown
[律所/公司抬头]
函号：[年份]第[XX]号

**律 师 函 / 侵 权 警 告 函**

致：[被警告方名称/姓名]

[律所名称/公司名称]受[委托人名称]（下称"委托人"）委托，就贵方涉嫌侵犯委托人知识产权事宜，郑重致函如下：

**一、委托人权利基础**
委托人系以下知识产权的权利人：
[商标注册号、类别、有效期限 / 专利号、类型、授权日期 / 著作权登记号]
（证明文件见附件1）

**二、贵方涉嫌侵权的事实**
经委托人调查取证：
[侵权产品/服务描述]
[侵权地点/平台/时间]
[侵权比对分析]
（证据材料见附件2）

**三、法律分析**
贵方的上述行为已构成对委托人[商标权/专利权/著作权]的侵害。依据《[商标法/专利法/著作权法]》第[X]条之规定......

**四、委托人的要求**
委托人现郑重要求贵方：
1. 自收到本函之日起[XX]日内，立即停止[制造/销售/许诺销售/使用/进口]上述侵权产品；
2. ......
3. ......

**五、后续措施**
如贵方逾期未履行上述要求，委托人将保留通过以下方式维护自身合法权益的权利：
- 向市场监督管理部门/知识产权主管部门投诉举报
- 向人民法院提起诉讼
- 向海关申请查扣侵权嫌疑货物（如涉及进出口）
- 向电子商务平台投诉举报（如涉及平台销售）

**六、声明**
委托人保留追究贵方全部法律责任的权利。贵方对本函有任何意见或协商意向，请于收到本函后[XX]日内与本所/本单位联系。

联系人：[姓名]
电话：[号码]
邮箱：[邮箱]
地址：[地址]

[单位名称]
[日期]
```

## 此技能不做什么

- 不实际发送函件。仅起草。由用户批准后按适当方式发送。
- 不在发送前绕过关卡检查。每次运行发送模式关卡。
- 在接收方不做终局性的权利判断。结构化的评估供律师决策。
- 不引用未被提供的法律条文（使用[核实]标注需要核实的具体条款编号）。
