# /ip-legal:invention-intake

**这是初步筛查，不是可专利性意见。**筛查绝不结论一项发明是可专利的——它结论该发明通过了初步筛查并值得现有技术检索和注册专利代理师审查，或需要补充信息，或命中了一项否定因素。现有技术检索是单独步骤；本技能不做。

## 中国法下职务发明的特殊考量

依据《专利法》第6条：执行本单位的任务或者主要是利用本单位的物质技术条件所完成的发明创造为职务发明创造。职务发明创造申请专利的权利属于该单位。

职务发明的认定：
1. 在本职工作中作出的发明创造
2. 履行本单位交付的本职工作之外的任务所作出的发明创造
3. 退休、调离原单位后或者劳动、人事关系终止后1年内作出的，与其在原单位承担的本职工作或者原单位分配的任务有关的发明创造
4. 主要是利用本单位的物质技术条件（资金、设备、零部件、原材料、不对外公开的技术资料等）所完成的发明创造

《专利法》第15条：对职务发明创造的发明人或者设计人，应当给予奖励；发明创造专利实施后，应当给予合理的报酬。

## 指示

1. 读取执业画像配置。如显示仅商标/著作权（无专利执业），说明并路由用户至别处。
2. 确认发明创造的性质——是否为职务发明？如是，本单位是否有明确的职务发明奖励制度？
3. 运行六项筛查。
4. 撰写发明筛查备忘录。
5. 底线裁决：PURSUE（值得现有技术检索+专利代理师审查）/INVESTIGATE（需要补充特定信息）/DECLINE（陈述具体理由）。永不说"可专利"。
6. 以决策树和关卡结束。

**此技能绝不结论一项发明是可专利的。**

## 中国《专利法》(2020)下的发明三类型

- **发明**：对产品、方法或者其改进所提出的新的技术方案（第2条）——保护期20年
- **实用新型**：对产品的形状、构造或者其结合所提出的适于实用的新的技术方案——保护期10年
- **外观设计**：对产品整体或者局部的形状、图案或者其结合以及色彩与形状、图案的结合所作出的富有美感并适于工业应用的新设计——保护期15年

## 六项筛查

### 筛查1：新颖性信号

《专利法》第22条第2款：新颖性，是指该发明或者实用新型不属于现有技术；也没有任何单位或者个人就同样的发明或者实用新型在申请日以前向国务院专利行政部门（CNIPA/国知局）提出过申请，并记载在申请日以后公布的专利申请文件或者公告的专利文件中。

现有技术：申请日以前在国内外为公众所知的技术（绝对新颖性标准）。

筛查要点：
- 发明人是否做了初步查新？
- 是否存在已知的竞争对手或研究团队在做类似方向？
- 是否有公开使用/发表/披露的记录？

### 筛查2：创造性/显而易见性标记

《专利法》第22条第3款：创造性，是指与现有技术相比，该发明具有突出的实质性特点和显著的进步（实用新型：实质性特点和进步）。

筛查要点：
- 技术方案是否在现有技术中能找到明确的启示？
- 区别技术特征是什么？
- "三步法"初步判断：(1)确定最接近的现有技术；(2)确定区别特征和实际解决的技术问题；(3)判断是否显而易见

### 筛查3：可专利主题资格

《专利法》第25条——不授予专利权的主题：
- 科学发现
- 智力活动的规则和方法
- 疾病的诊断和治疗方法
- 动物和植物品种（但其生产方法可授权）
- 原子核变换方法以及用该方法获得的物质
- 平面印刷品的图案、色彩或者二者的结合作出的主要起标识作用的设计

《专利法》第5条：违反法律、社会公德或者妨害公共利益的发明创造不授予专利权。

还应检查：是否涉及计算机程序（软件专利需要"技术方案+技术问题+技术效果"）？是否涉及商业方法？

### 筛查4：公开/期限日

《专利法》第24条——不丧失新颖性的宽限期（6个月）：
- 在国家出现紧急状态或者非常情况时为公共利益首次公开
- 在中国政府主办或者承认的国际展览会上首次展出
- 在规定的学术会议或者技术会议上首次发表
- 他人未经申请人同意而泄露其内容

重要时间节点：
- 首次公开日（如有）
- 宽限期截止日（公开日+6个月）
- 潜在竞争对手动态

### 筛查5：可检测性

- 产品权利要求：是否在最终产品上可以检测到侵权？
- 方法权利要求：是否有办法证明他人的制造方法使用了本专利方法？（新产品制造方法的举证责任倒置——第66条）
- 维权可操作性

### 筛查6：战略价值

- 是否与公司核心业务方向一致？
- 是否是竞争对手也在积极布局的技术领域？
- 是否有海外申请需求？（PCT/巴黎公约途径）
- 是从头开始布局还是防御性公开？

## 护栏

**永不说"可专利"。**"通过初步筛查，值得进一步调查。"

**绝不在此技能中做现有技术检索。**交由专业检索机构或专利代理师完成。

**在边界案件上延期判断。**对任何主题资格、创造性存在边界情况的情形，标记为需要专家审查，不做及格/不及格结论。

## 此技能不做什么

- 不替代注册专利代理师的专业判断。
- 不进行现有技术检索。
- 不撰写专利申请文件（权利要求书/说明书/附图）。
- 不对最终是否能获得授权做任何保证。
