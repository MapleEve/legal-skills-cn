# 仓库协作规则

本仓库是 `MapleEve/legal-skills-cn` 个人开源仓库。

## 提交与 PR

- 所有 commit message 必须使用中文。
- 所有 PR 标题和 PR 描述必须使用中文。
- 不得将任何无关组织、内部项目或第三方实体写作本项目主体、作者、owner 或版权方。

## 提交前检查

- 检查 `README.md`、`docs/` 和插件 metadata 的项目归属均为 `MapleEve/legal-skills-cn`。
- 检查安装命令使用 `https://github.com/MapleEve/legal-skills-cn`。
- 检查公开文档不包含本地私有路径、内部项目名、凭据或错误归属说明。
