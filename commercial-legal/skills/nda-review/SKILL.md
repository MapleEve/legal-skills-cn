# 保密协议审查

## 前置参考资料

1. 先读取 `../../references/contract-law-core.md`，再进行中国法保密协议条款审查和风险标注。

## 功能目的

中国法下保密协议审查。核心法律依据：《民法典》合同编、《反不正当竞争法》第9条（商业秘密保护）。区分单方保密协议与双方保密协议，适用不同审查标准。

## 审查工作流

### 步骤1：识别保密协议类型

在审查条款前，判断：

1. **单方保密协议**：仅一方披露、另一方接收保密信息
2. **双方保密协议**：双方均可能互相披露保密信息（对等/双向）

**类型识别规则：**
- 标题为"保密协议"或"商业秘密保护协议"
- 区分单方/双方关键看：保密义务是否为单向。若仅约定一方保密义务为单方；约定双方互相承担保密义务为双方
- 若不明确，询问："该保密协议是单方（仅你方向对方披露保密信息）还是双方（互相披露）？"

### 步骤2：保密信息范围审查

审查保密信息定义，与《反不正当竞争法》第9条商业秘密定义做一致性检查。

**《反不正当竞争法》第9条商业秘密三要件：**
- 不为公众所知悉（秘密性）
- 具有商业价值（价值性）
- 经权利人采取相应保密措施（保密性）

**审查要点：**

| 审查项 | 绿灯标准 | 黄灯触发 | 红灯触发 |
|---|---|---|---|
| 保密信息范围 | 合理界定（如"明确标记为保密的信息"或合理列举） | 定义偏宽但可协商 | "全部披露信息"均为保密——范围过宽，实质上等同于将日常沟通均纳入保密义务 |
| 口头/视觉信息的书面确认 | 有合理确认窗口（如30天） | 窗口过短（如7天） | 无书面确认要求——口头信息永久保密不可接受 |
| 保密期限 | 约定明确期限（如合同期内及终止后2-5年） | 终止后超过5年 | 永久保密 |

**特别注意：**
- 保密信息定义不得与《反不正当竞争法》第9条冲突——不能将"已为公众所知悉的信息"也强行纳入保密范围
- 格式条款中不合理扩大保密范围的约定可能依据《民法典》第497条被认定无效

### 步骤3：保密义务期间审查

检查保密义务的起止时间：

1. **保密期间**：合同有效期内的保密义务
2. **保密存续期**：合同终止/解除后，保密义务是否继续以及存续多久
3. **商业秘密特殊保护**：若涉及商业秘密，《反不正当竞争法》第9条的保密义务在商业秘密公开前持续有效，不受合同期限限制

**审查要点：**
- 合同终止后保密存续期是否合理（通常2-5年）
- 商业秘密是否享有单独的、不受期限限制的保护
- 注意：约定了过短的保密存续期可能导致商业秘密在合同终止后失去合同保护

### 步骤4：违约责任审查

**法律依据：**
- 《民法典》第577条：违约责任一般规定
- 《民法典》第584条：损失赔偿额的可预见规则
- 《民法典》第585条：违约金过高可请求调减（超过实际损失30%视为"过分高于"）
- 《反不正当竞争法》第17条：侵犯商业秘密的损害赔偿额计算规则

**审查要点：**

| 审查项 | 绿灯标准 | 黄灯触发 | 红灯触发 |
|---|---|---|---|
| 违约金金额 | 约定合理违约金或按实际损失计算 | 约定较高违约金但可协商 | 违约金明显过高（预估超过实际损失30%）|
| 损失赔偿范围 | 包括直接损失+合理维权费用 | 包括律师费 | 不限范围的"全部损失" |
| 赔偿上限 | 合理上限或按法定 | 无上限但非重大交易 | 无上限 + 范围过宽的保密义务 |

**特别提醒：**《民法典》第585条赋予当事人请求人民法院或仲裁机构调减过高违约金的权利。违约金过高的举证责任在请求调减方。

### 步骤5：合同包含非保密义务的审查

保密协议可能隐藏超出保密范围的条款。检查是否包含：

- 竞业限制条款（需审查与《劳动合同法》第23-24条一致性）
- 禁止招揽条款
- 排他性合作条款
- 知识产权转让/许可条款
- 优先购买权
- 最惠国待遇条款
- 宽泛的仲裁/管辖条款（管辖范围超出保密争议）

**若发现上述任一条款，自动标记黄灯：**
> 本文件名为保密协议，但包含[具体条款类型]。其实际范围超出保密协议的正常范畴，建议路由至律师审查。

### 步骤6：争议解决条款审查

**法律依据：**
- 《民事诉讼法》第35条：协议管辖须与争议有实际联系
- 《仲裁法》第16条：仲裁协议须明确约定仲裁机构
- 诉讼与仲裁二选一，不得同时约定

**审查要点：**

| 审查项 | 绿灯标准 | 黄灯触发 | 红灯触发 |
|---|---|---|---|
| 争议解决方式 | 诉讼或仲裁，明确二选一 | 约定方式不明确 | 同时约定诉讼和仲裁（无效） |
| 管辖法院 | 与争议有实际联系（《民诉法》第35条） | 约定地无实际联系 | 约定对己方明显不利的管辖地 |
| 仲裁机构 | 明确指定仲裁委员会 | 仲裁规则不明 | 未指定具体仲裁机构（可能无效） |
| 法律适用 | 中国大陆法律 | 其他法域法律 | 排除中国法律适用且与合同无实际联系 |

### 步骤7：分类输出

#### 绿灯——可签署

保密协议无明显法律风险，符合审查标准。

```markdown
## 保密协议审查：[合同相对方名称]

🟢 绿灯——可签署

### 审查结论
经审查，本保密协议不存在需标记的法律风险。保密信息范围合理、保密期限适当、
违约责任约定公平、争议解决条款有效、与《反不正当竞争法》第9条商业秘密定义一致。
按标准流程签署。

### 审查明细

| 审查项 | 状态 | 法律依据 / 说明 |
|---|---|---|
| 保密协议类型 | [单方/双方] | — |
| 保密信息范围 | 通过 | 《反不正当竞争法》第9条 |
| 保密期限 | 通过 | 约定期限合理 |
| 保密存续期 | 通过 | 终止后[X]年 |
| 违约责任 | 通过 | 《民法典》第577-585条 |
| 争议解决 | 通过 | 《民诉法》第35条 / 《仲裁法》第16条 |
| 非保密义务检查 | 通过 | 未发现超出保密范围条款 |
| 格式条款 | 通过 | 《民法典》第496-498条 |

### 后续步骤
1. 按标准用印流程签署（填写用印审批单）
2. 签署后原件存档（至少一份），扫描电子档归档
3. 如需，在合同台账中登记
```

#### 黄灯——需律师审查特定事项

一项或多项条款存在需关注的问题，但不构成签约障碍。

```markdown
🟡 黄灯——标示事项需法务审查

### 审查结论

### 标示事项

   **内容**：[一句话说明条款内容]
   **标示原因**：[具体法律风险——引用法律依据]
   **法律风险**：🔴红线 / 🟠重大 / 🟡一般 / 🟢提示
   **业务影响**：[对业务的实际影响]
   **建议方案**：[接受 / 修改措辞 / 协商调整]

[依次列出各标示事项]

### 其余通过项

| 审查项 | 状态 |
|---|---|
| [各项通过审查项] | 通过 |

### 后续步骤
1. 将标示事项与法务确认
2. 法务确认后可签署
3. 签署后按标准归档流程处理
```

#### 红灯——暂缓签署，需法务深入审查

保密协议存在重大法律风险，不建议在解决前签署。

```markdown
## 保密协议审查：[合同相对方名称]

🔴 红灯——存在重大法律风险，暂缓签署

### 审查结论
本保密协议存在以下重大法律风险。建议在问题解决前暂缓签署。

### 重大风险事项

**1. [问题标题]** — 第[X]条
   > "[原文引用]"
   **法律风险分析**：[具体说明违反何法律规定 / 存在何种风险]
   **法律依据**：[引用具体法条]
   **建议回应**：[改用我方模板 / 以具体措辞反驳 / 重新协商]

[依次列出各重大风险]

### 后续步骤
1. **不要**告知对方我们打算签署
2. **不要**提交用印审批
3. 将本审查意见提交法务总监/合伙人审查
4. 由法务决定是否与对方协商修改还是放弃签署
```

## 合同相对方背景考量

**大型企业/国有企业的保密协议模板：**
通常采用其标准模板、谈判空间有限。审查时区分"形式性偏离"与"实质性风险"——模板措辞的差异不一定等同于法律风险。若业务关系重要，重点判断实质性风险而非文风差异。

**初创企业/中小企业的保密协议：**
通常谈判空间较大。若对方的保密协议存在问题，最快路径往往是提议使用我方标准模板。

**涉外保密协议：**
注意法律适用和管辖约定——若约定适用外国法律或外国法院管辖，需额外审查是否符合《民事诉讼法》第35条关于协议管辖的规定，并评估跨境执行的实际可行性。

## 常见陷阱

1. **保密信息范围过宽**："与本合同相关的全部信息"——将日常沟通纳入保密范围
2. **永久保密期限**：约定"永久"保密可能与《民法典》公平原则冲突，且对于非商业秘密信息属不合理限制
3. **过高违约金**：超过实际损失30%的部分可能被法院调减（《民法典》第585条）
4. **同时约定诉讼和仲裁**：属无效管辖条款
5. **单方保密协议中的不对等义务**：需确认单方保密在本商业场景下是否合理
6. **隐藏的非保密承诺**：保密协议中包含竞业限制、禁止招揽等条款

## 注意事项

- 本技能用于保密协议审查分类，不代替律师出具正式法律意见
- 红灯事项必须由法务律师或合伙人最终判断
- 涉及重大金额或跨境交易的保密协议，建议由合伙人对审查结论进行复核
- 合同相对方为大企业时，模板性偏离不应自动触发红灯
